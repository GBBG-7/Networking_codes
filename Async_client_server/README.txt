This folder contains two simple python scripts that demonstrate how 
the client-server model works.

[I used the python asyncronouse module to write this script ]

NOTE: Make sure you have python installed on your machine

Make sure you run the server before the client

Firstly,To execute these script,open two terminals or tabs,
run python ASYNC_SERVER.py which is the server,specify the host 
and port number to listen on and the server will be created listening on 
host and port,when data is sent to the server,it will be printed in the 
terminal and echoed back to the client.
The server is going to run untill explicitly interrupted

Secondly,run python ASYNC_CLIENT.py, specify the host to connect, 
and the port number and the data to be sent,when you click enter,the data you 
wrote will be sent to the server and closed,the client must received the data sent
because the server echoes the data back
